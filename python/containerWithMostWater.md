## 求出最大可容纳的水量
---
给出一个正数list，比如a1,a2,a3,a4...an，每个元素代表二维坐标轴上的一个点，比如a5代表点（5，a5）。从这些坐标点出发，做出垂直于X轴的线，其中任意两条线与X轴构成一个容器，求出可以容纳的最大水量。

**容器不能倾斜，并且list至少包含两个元素**

### Example:
输入: [9,6,10,5,3,7,10,8]

输出: 56

**注意: "aba"同样为正确答案。**


	class Solution(object):
	    def maxArea(self, height):
	        """
	        :type height: List[int]
	        :rtype: int
	        """
	        i = 0
	        j = len(height) - 1
	        result = -1
	        while i < j:
	            if height[i] < height[j]:
	                res = (j - i) * height[i]
	                i = i + 1
	            else:
	                res = (j - i) * height[j]
	                j = j - 1
	            if res > result:
	                result = res
	        return result
	
	
	
	if __name__ == '__main__':
	   solution=Solution()
	   print solution.maxArea([9,6,10,5,3,7,4,8])

首先想到的最简单粗暴的方法，穷举法，将所有的情况一一列举出来，求出最大可容纳的水量。不过性能糟糕，时间复杂度为O(n^2)。

其实在解题过程中也在思考，可以去掉一些冗余的计算，降低时间复杂度。但是最终没能思考出一个很好的方法。

在Discuss中找到巧妙的方法，时间复杂度为O(n)，思路如下：

1、从最外围开始，最先两根线选定为①和⑧，此时可容纳水量为(8-1)*8=56。第一根线①的长度为9，第二根线⑧的长度为8，第二根线比第一个线短，那么无论第一根线无论如何左移，可容纳水量只能是<=56.

2、因此只能移动第二根线，此时两个线选定为①和⑦，此时可容纳水量为(7-1)*9=42。第一根线①的长度为9，第二根线⑦的长度为10，第二根线比第一根线长，同理，无论第二根线无论如何左移，可容纳水量只能是<=42.

3、因此只能移动第一根线，此时两个线选定为②和⑦，此时可容纳水量为(7-2)*6=30。第一根线②的长度为6，第二根线⑦的长度为10，第二根线比第一根线长，同理，无论第二根线无论如何左移，可容纳水量只能是<=30.
